Physicolize

Physicolize is a physics simulation application developed by the Software Department of Attila Robotics within the scope of STEAM (Science, Technology, Engineering, Art, Mathematics). It is designed to support high school students, particularly in simulating fundamental mechanics topics including Newton's Laws at the high school level. With Physicolize, students can perform simulations of the physics topics they learn, helping them grasp the logic of the subject matter. User-friendly Physicolize offers a wide and varied range of simulations, where users are free to create any simulations they want without having to adhere to the physical laws of the real world. They can modify numerous physical components affecting an object or environment, thus experiencing some simulations that would not occur in the real world. Currently, Physicolize is only compatible with Windows operating systems. You can download the built version of Physicolize from the following link. Physicolize is developed using Unity and utilizes the PhysX physics engine by Nvidia, which is used by Unity

Application Mechanic

Physicolize provides a user with an object placement system. With this system, a selected object can be placed in the desired rotation, size, and position. Additionally, when certain variations are provided, it has an automatic placement system that connects objects to each other. For example, an object can be connected to a chain, the chain to a holder, and the holder can be attached to the ceiling.

Comprehensive Interface

Physicolize offers users a comprehensive interface and explains the function of each panel in this interface. Users can observe the changes in the physics of the object by applying the operations in these panels to the object.

3 dimensional

Physicolize, unlike other physics simulations, has a three-dimensional structure, which allows it to offer simulations similar to those in the real world. Not only its three-dimensional aspect but also many operations in its interface are elements that make it unique.
